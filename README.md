# Toolkit_for_Linux

Toolkit_for_Linux 是一个由我开发的面向小白的Linux工具集，旨在简化上手难度，涵盖了多种实用功能。

---

## 注意事项
> 由于本人精力有限且主力系统是Fedora,故只开发了用于Fedora的脚本（部分脚本可以用于其他发行部）
---

## 使用方法
1. 克隆此仓库：
   ```bash
   git clone https://github.com/XianYin69/Toolkits_for_Linux.git
   ```
2. 使用脚本 :
   ```bash
   chomod +x PATH/*
   bash ./*.sh
