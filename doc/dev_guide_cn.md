## 标准
### 文件结构
-   Lazy_Linux(主目录)   
    -   lazy_linux.sh（主程序）    
    -   LICENSE（许可证）  
    -   README（说明）
    -   lib （“库”）
        - lib_name  （“库名”）
            - lib_main.sh （“库函数文件”）
            - sub-lib （“子库名”）
                -   sub-lib.sh （“子库函数文件”）
    -   sample（示例）
    -   src（资源）
        - source （资源名）
            -   source.xxx（资源文件）
    -   var（变量）
        -   variable  （变量名）
            -   variable.txt（变量文件）
---
